About ColorIt

ColorIt is an automatic B/W and greyscale image colorization tool that uses Deep learning and Convolutional Neural Networks to colorize the input image.

The model is trained on nearly 60,000 images from ImageNet and uses Google's pre-trained feature extractor for feature extraction.

This project is submitted as a part of the course EEL 6761 Cloud Computing and Storage at the University of Florida, Fall'17.
